# TextConvertToQDos

Text converter to qdos character set.


Sintax: 

```
   Textq sourceFile destfile [options]
```
   

Examples: 

```
   Textq myWinFile.txt myQDosFile_txt
   Textq myUtf32File.txt myQDosFile_txt -UTF32
```

Options: 

```
   -UTF8  (default value)   
   -Unicode   
   -ASCII   
   -UTF32
```
